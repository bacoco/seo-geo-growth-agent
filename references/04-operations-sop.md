# 04 — Operations SOP

## Week 0: Foundation

Goal: make the site measurable, crawlable, indexable, and ready for content production.

- Fill project variables.
- Complete owner checklist for GSC, GA4, site access, CTA URL, and data exports.
- Audit robots.txt, sitemap, canonicals, indexability, and analytics.
- Create the initial 30–50 keyword seed list.
- Build the first keyword-to-page map.
- Record baseline: indexed URLs, organic sessions, conversions, tracked keyword distribution, AI-source sessions if available.

## Week 1: BOFU pages

Goal: cover high-intent decision queries.

- Publish or refresh product, pricing, alternatives, and comparison pages.
- Add FAQ and schema to BOFU pages.
- Add strong CTA blocks with UTM parameters.
- Submit/push new canonical URLs.
- Start tracking rankings and GSC queries.

## Week 2: Internal links and distribution

Goal: strengthen the page graph and expand discovery.

- Build topic clusters around the most valuable category pages.
- Add internal links from high-impression/high-authority pages to target pages.
- Audit orphan pages.
- Syndicate carefully only when canonical support is available and the owner approves.
- Refresh sitemap and indexing queue.

## Week 3: GEO reinforcement

Goal: make pages easier for AI engines to understand and cite.

- Add direct answer blocks to high-priority pages.
- Add key facts, comparison, and decision tables.
- Add or fix JSON-LD.
- Add last-updated dates and author/entity trust signals.
- Run AI citation checks for target queries.
- Fix inaccurate entity descriptions.

## Week 4: CTR and scale

Goal: turn impressions into clicks and clicks into conversions.

- Find high-impression low-CTR pages.
- Rewrite titles and meta descriptions.
- Improve pages ranking 11–20 with internal links, content gaps, and schema.
- Review CTA performance by page type.
- Decide which templates are safe to scale.

## Daily loop

1. Update tracked keyword distribution.
2. Check indexation and sitemap anomalies.
3. Review GSC/GA4 changes when fresh data exists.
4. Check AI-source sessions and citation observations.
5. Pick 1–3 actions that are likely to move rankings, citations, CTR, or conversions.
6. Record owner blockers.

## Weekly loop

- Import new GSC queries.
- Add queries to keyword map or merge with existing intent.
- Review high-impression low-CTR opportunities.
- Review pages ranking 11–20.
- Add internal links.
- Review CTA funnel performance.
- Check AI citation accuracy.

## Monthly loop

- Refresh keyword volumes and difficulty.
- Expand tracked keyword set.
- Review cannibalization.
- Refresh outdated pages.
- Update comparison pages.
- Rebuild priority roadmap.
- Review content templates for quality before scaling.
