# Owner Data Request

Target site: `{{site}}`

Use this template to ask the site owner for data that cannot be proven from public HTML, screenshots, or public measurement tools.

## Requested Exports

| Source | Export | Why |
|---|---|---|
| Google Search Console | Queries, pages, countries, devices, impressions, clicks, CTR, average position, indexing issues | Search visibility and CTR proof |
| GA4 | Landing pages, source/medium, conversions, engagement, AI-assistant referrals | Visit and conversion proof |
| Bing Webmaster Tools | Search performance, crawl/indexing issues, IndexNow, AI Performance if available | Bing/Copilot and IndexNow proof |
| Server logs | Bot/user-agent logs for search, AI search, training, and user-triggered fetchers | Crawler access and WAF proof |
| Cloudflare Analytics | Traffic, cache, WAF/firewall, bot, and response-status analytics | CDN/cache and blocked-request proof |

## Paid Tool Consent

Default policy: ask before using Haloscan, Semrush, Ahrefs, DataForSEO, Similarweb, SerpApi, or any tool that consumes paid credits.

## Cloudflare Rule

Do not bypass Cloudflare. Use owner-approved analytics, logs, cache headers, WAF events, or temporary owner-approved access.
